# FSO-Zigbee-RPi-Projects

## Overview
This repository contains Raspberry Pi based communication projects involving:

- Free Space Optics (FSO)
- Zigbee Communication
- Hybrid FSO-Zigbee Communication

## Projects Included

### 1. FSO on Raspberry Pi
Contains:
- ASCII transmitter and receiver
- Image transmitter and receiver along with the transmitted image
- Documentation

### 2. Zigbee on Raspberry Pi
Contains:
- ASCII transmitter and receiver
- Image transmitter and receiver transmitted image
- Documentation

### 3. Hybrid FSO-Zigbee
Contains:
- ASCII transmitter and receiver
- Image transmitter and receiver transmitted image
- Documentation

## Technologies Used
- Raspberry Pi
- Python
- Zigbee Modules
- Optical Communication
- Wireless Communication

Note: Before using and executing the codes, verify the path of the image which we are transmitting as it differs a bit from the original code
